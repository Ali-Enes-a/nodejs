Node.js Nasıl Kurulur?

Node.js kurulumu için nodejs.org adresine giriyoruz. Downloads sayfasında LTS ve Current isminde iki farklı sürüm bizi karşılıyor LTS ve Currnet isminde bunlardan kararlı sürüm olanı LTS sürümüdür fazla hata almamak için bu sürümü indiriyoruz.

LTS: Uzun süreli destek sürümü.

Current: En son özellikleri destekleyen sürümdür.

İndirdiğimiz sürümün kurulum dosyasını çalıştırıp yüklüyoruz.

Kurulumdan sonra cmd de 'node -v' veya 'node --version' komutlarını girerek kurmuş olduğumuz sürümü görüntüleyebilmekteyiz.



Visual Studio Code Kurulumu

Windows

VS Code'u Windows işletim sistemine kurmak için;

  1. VS Code Kurulum Dosyası'nı bilgisayarımıza yüklüyoruz.
  2. Kurulum dosyasını çalıştırıyoruz ve yönergeleri takip ederek kurulumu tamamlıyoruz.
  3. Varsayılan olarak, VS Code C:\users\{kullanıcı adı}\AppData\ Local\Programs\Microsoft VS Code yoluna kurulur.
  4. VS Code'u başlatmak için Başlat menüsünü kullanabiliriz yada komut istemcisinden code . komutunu çalıştırabiliriz.


macOS

  1. VS Code Kurulum Dosyası'nı bilgisayarımıza indiriyoruz.
  2. Çift tıklayarak sıkıştırılmış dosya paketini açıyoruz.
  3. Visual Studio Code.app'i, Uygulamalar (Applications) klasörüne sürükleyerek Launchpad'de kullanılabilir duruma getiriyoruz

İhtiyaç halinde VS Code'u, terminalden code komutunu çalıştırarak başlatabiliriz. Bunun için:


  1. VS Code başlatıyoruz.
  2. Command+Shift+P kısayolunu kullanarak Komut Paleti (Command Palette)'i açıyoruz ve "shell command" yazıyoruz.
  3. Açılan menüden Shell Command: Install 'code' command in PATH komutunu seçiyoruz.
  4. Yeni $PATH değerinin geçerli olması için açık olan terminal penceremizi yeniden başlatıyoruz. Artık, terminal penceresinde code . komutunu çalıştırarak bulunduğumuz klasörün VS Code'a açılmasını sağlayabiliriz.

Linux


VS Code herhangi bir Linux dağıtımına kurmak için kullanabileceğimiz en basit ve hızlı yöntem, Snap'tir. Daha önce kullanmayanlar için Snap; Linux kullanıcılarına özel olarak oluşturulmuş bir uygulama yükleme platformudur. Snap paketi olarak hazılanmış milyonlarca uygulamayı, hemen hemen tüm Linux dağıtımlarına kolayca yükleyebilmemize olanak sağlar.


VS Code, Microsoft tarafından resmi olarak Snap Store'da bir Snap paketi olarak dağıtılmaktadır.


VS Code Snap paketini kolayca yüklemek için bir Linux terminalinden şu komutu çalıştırıyoruz:

sudo snap install --classic code

Kurulum tamamlandıktan sonra Snap, arka planda VS Code'u otomatik olarak güncellemeleri denetleyecek. Yeni bir güncelleme olduğunda, bizi bilgilendirecek ve onay verirsek yükleyecek.

Postman Nedir?

Postman backend apilerini test emek için oldukça kullanışlı bir rest client’tır. Eğer elinizde var olan bir API’yi hızlıca test etmek sonuçlarını incelemek isterseniz Postman kullanıcı dostu arayüzü ile işlerinizi oldukça kolaylaştırıyor. Hazırladığınız request’leri export edip arkadaşlarınızla da paylaşabiliyorsunuz. Aynı zamanda requestler arası veri aktarabiliyor, bir api’nin çıktısını diğerinin outputu olacak şekilde geliştirme yapabiliyorsunuz. Çok az Javascript bilgisi ile hızlıca testler yazabiliyor, yazdığınız testleri kaydedip sonra tek bir tık ile çalıştırabiliyorsunuz.


Service Nedir?

Servisler platformdan bağımsız olmak üzere birçok uygulama yada cihazın birbiri ile iletişim kurmalarını sağlayan yapılardır.


Soap Nedir?

İnternet üzerinden küçük miktardaki verileri yada mesajları aktarma protokolüdür. SOAP mesajları XML formatındadırlar genellikle HTTP protokolü bazen de TCP/IP protokolü kullanılarak gönderilir.


REST Nedir?

REST web servis oluşturmak için kullanılan web servis yöntemlerinden birisidir. Programlama dilinden bağımız olarak web servis oluşturmak için kullanılan bir yapıdır. HTTP protokolüne ait HTTP yöntemlerini ve HTTP durum kodlarını kullanarak gelen isteklere XML, JSON, CSV vb. şekilde cevap verilir.


RESTful Nedir?

RESTful web servisler REST mimarisini temel alarak geliştirilmiştir. Client-Server arasındaki veri platform bağımsız ve az yükle yapılır. Response olarak XML ve JSON kullanılır.
